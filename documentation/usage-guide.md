# Step-by-Step Guide

## 1. Create an Account
1. Visit the Algonive chat URL.
2. Click **Request access**.
3. Provide your name, work email, and a secure password.
4. Submit to receive immediate access.

## 2. Sign In
1. Navigate to the login page.
2. Enter the same work email and password.
3. Optional: toggle **Keep me signed in**.
4. Press **Enter workspace**.

## 3. Onboarding Overview
- Left navigation shows direct chats and groups.
- Right side displays the currently selected room.
- Top header shows your presence status and quick actions.

## 4. Start a Direct Chat
1. In the sidebar, click **New Chat**.
2. Choose **Direct** and enter one or more colleague emails.
3. Submit to create the room; it opens automatically.

## 5. Launch a Group Room
1. Click **New Chat** and select **Group**.
2. Provide a room name plus comma-separated member emails.
3. Submit to create and join the group room instantly.

## 6. Send Messages & Files
1. Type into the composer at the bottom of the chat window.
2. Attach files using the upload icon (images preview inline).
3. Press Enter or the send button to post.

## 7. Real-Time Collaboration
- Typing indicators show when teammates are responding.
- New messages appear instantly via sockets.
- Push notifications alert you even when the tab is hidden (browser permission required).

## 8. Edit or Delete Messages
1. Hover or tap the 3-dot menu on your own message.
2. Choose **Edit message** or **Delete for me**.
3. Edits mark the bubble with (edited). Deletes convert to a placeholder.

## 9. Enable Desktop Notifications
1. When prompted, allow browser notifications.
2. Ensure service worker registration succeeded (DevTools > Application).
3. Notifications open the room that generated the alert.

## 10. Log Out Securely
1. Use the **Sign out** button in the top navigation.
2. Local tokens are cleared and the socket is disconnected.
3. Close the browser tab when finished.

Happy collaborating!
